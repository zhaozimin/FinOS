①建账户 · 进「财务设置 → 账户管理」，加你常用的卡 / 微信 / 支付宝。区分"工作 / 生活"归属，资产 vs 负债。
②填当前余额 · 直接输入卡里现在有多少钱（不是初始余额）。差额会自动记成"余额调整"交易。

Phase 1

基建打底

资金状况 + 数据底座 — 先把"账户怎么算"和"附件怎么挂"做对，后面所有功能都建立在这上面。

P1 · 1.5

资产 / 负债 + 信用卡额度

账户分两类：资产（卡里的钱、零钱）和负债（信用卡欠款、贷款）。净资产 = 资产 − 负债，会显示在「财务状况」顶部 KPI。

💳
招商信用卡 负债
已用 ¥3,820 / ¥30,000 · 可用 ¥26,180
上手三步
  1. 进「财务设置 → 账户管理」，新增账户。
  2. 「性质」选负债（信用卡 / 贷款）；类型选信用卡。
  3. 填「信用额度」（如 ¥30,000）和「当前已欠」（如 ¥3,820）→ 卡片底部出现进度条。
P1 · 2.6

余额调整审计 — 黑洞资金

用户在设置页改"当前余额"时，系统会自动写一条「余额调整」交易补差。意思是：你直接说"现在卡里有 ¥1234"，但实际记账推算出 ¥1180，那 ¥54 的差额会被记成 income/expense，带 余额调整 · 黑洞资金 标签，永不被吞掉。

这套机制让"账实不符"变成可见 + 可追溯的交易，而不是悄悄改 openingBalance。
上手三步
  1. 进「财务设置 → 账户管理」，找一个账户，改「当前余额」字段。
  2. 右上角点「保存设置」。
  3. 切到「资金流水」，搜索 余额调整 看到自动生成的交易。
P1 · 1.6

附件 + 👁 眼睛图标

每条交易可以挂多张图片 / PDF（单张 ≤ 10 MB），如发票照、收据扫描。流水表行末有眼睛图标，点击全屏查看（支持 ← → Esc）。

上手三步
  1. 「资金流水」点任意一条交易 → 弹出「编辑交易」。
  2. 在「附件 / 发票」区点「添加附件」上传图片。
  3. 保存后回流水列表，行末眼睛图标出现，点击查看。
P1 · 2.4

客户 / 合作方名册

登记长期合作的客户、供应商、雇主。登记后在交易上挂"对手方"字段，方便回款查询和归账。

上手三步
  1. 进「财务设置 → 客户与合作方」，点新增对手方。
  2. 填名称、类型（客户 / 供应商 / 雇主 / 其他）、默认结算账户。
  3. 下次记账时在「对手方 / 客户」下拉选中，自动归账。
Phase 2

日常记账增强

把"减少手输 + 个人理财核心"做扎实 — 预算、周期账目、储蓄目标、账单一键导入。

P2 · 1.1

预算管理

给支出分类设月度预算，超过 80% 变黄、超过 100% 变红。「财务状况」页有总览，「资金流水」筛选支出时顶部还有 mini 进度条。

外卖饮食 86.4%
已花 ¥432 / ¥500 剩余 ¥68
上手三步
  1. 进「财务设置 → 类别管理」，找一个支出分类。
  2. 在「月度预算」字段填金额（如 ¥500），保存。
  3. 切到「财务状况」首屏，看「预算进度」卡片。
P2 · 1.2

周期性交易（订阅 / 房租 / 工资）

按日 / 周 / 月 / 年自动生成交易。设了"每月 1 号 ¥1500 房租"后，每月 1 号系统会自动写一条 expense。每次刷新页面都会扫一次到期规则（30 秒 debounce）。

上手三步
  1. 进「财务设置 → 周期账目」，点「新增规则」。
  2. 填名称（如 "Netflix"）、金额、类型、账户、分类、频率（每月 / 每周等）。
  3. 设「下次触发」为今天或之前，保存后 → 立刻生成第一条交易。
生成的交易带 source='recurring' + 周期账目 标签，可在「资金流水」搜索过滤。
P2 · 1.3

储蓄目标

给项目设目标金额 + 截止日期，财务状况首屏显示进度圆环：< 30% 蓝、30-60% 黄、≥ 60% 绿。逾期未达目标显示红色警告。

65%
买台新电脑
¥6,500 / ¥10,000 · 剩 32 天
上手三步
  1. 进「财务设置 → 项目管理」，找或建一个项目。
  2. 展开「储蓄目标」折叠区，填目标金额 + 目标日期。
  3. 可选：选「资金来自账户」，让进度按账户余额算（否则按项目下的 income 累加）。
P2 · 1.4

账单导入（微信 / 支付宝 / 招行 / 通用 CSV）

三步向导：选模板 + 上传 → 预览编辑 → 提交。系统按列名智能识别（日期 / 金额 / 对方 / 备注），并按你账户和分类的 keywords 自动匹配。

上手三步
  1. 进「财务设置 → 账单导入」，选模板（首次推荐"通用 CSV"），上传文件。
  2. 预览表会展示解析结果，已自动匹配的分类 / 账户可勾选 / 编辑 / 跳过。
  3. 点「确认导入」批量入库，结果弹窗显示成功 / 失败数。
支持 GBK / UTF-8 / GB18030 自动识别，国内银行流水基本不用转码。
Phase 3

创业者业务

项目预算 + 发票 + 税务 + 项目 P&L — 把"经营记账"四件套做齐。

P3 · 2.1

项目预算 — 实心 vs 描边双柱

给项目设「预算成本」+「期望收入」后，「财务状况」的项目柱状图会切到 4 柱模式：实心 = 实际，描边虚心 = 预算。tooltip 显示完成度百分比。

公司运营
A 客户项目
B 客户项目
实心 = 实际 · 描边 = 预算 / 期望
上手三步
  1. 进「财务设置 → 项目管理」，展开任意项目的「项目预算 / 期望」折叠区。
  2. 填预算成本 + 期望收入 + 起止日期。
  3. 切到「财务状况」看双柱图，hover 看完成度。
P3 · 2.2

发票追踪

交易上有「已开 / 应开发票」开关。开了之后从已上传的附件里绑定一张作为"发票"。"应开但没绑"的会在流水表行末显示橙色"缺"小徽章 — 提醒你尽快补开。

上手三步
  1. 编辑一条 income 交易，勾选「已开 / 应开发票」。
  2. 先上传发票图片到「附件 / 发票」区。
  3. 下方"发票"区从下拉里选这张附件作为发票绑定。保存。
P3 · 2.3

税务字段 + 报税导出

每条交易可选「税务分类」：个人 / 经营收入 / 可抵扣支出 / 不可抵扣 / 内部往来。一键导出 5 sheet 的报税 Excel：业务收入 / 可抵扣 / 不可抵扣 / 汇总（含预估增值税 + 个税 + 社保）/ 说明。

上手三步
  1. 进「财务设置 → 税务设置」，调整税率（默认中国小规模 / 个体户：3% 增值税 / 6 万年度起征点 / 20% 个税 / 10% 社保）。
  2. 编辑交易时为业务相关的项设「税务分类」（如 经营收入 / 可抵扣支出）。
  3. 回「税务设置」面板，选年度 + 季度，点「下载报税表」即得 xlsx。
⚠️ 预估算法是简化版，与真实税法可能有差异。请以专业税务人员意见为准。
P3 · 2.5

项目 P&L 报表

每个项目的「查看 P&L」按钮（柱状图标）打开 720px 右抽屉：4 个 KPI（实际成本 / 实际回款 / 净利润 / 利润率）+ 月度净利润折线 + 项目所有相关流水。

上手三步
  1. 进「财务设置 → 项目管理」。
  2. 找一个项目卡片，点击右上角的📊 柱状图标。
  3. 抽屉打开后查看 KPI、折线、流水。Esc 关闭。
Phase 4

全局视图

多币种 + 现金流预测 + 全局搜索 + 仪表盘可定制 — 把整个系统装上"上帝视角"。

P4 · 1.8

现金流预测 + 月度订阅

基于「周期账目」(P2-1.2) 推算未来 30 / 60 / 90 天的累计净流入折线，画在「财务状况」首页。旁边的「月度订阅」卡片按金额倒序列出所有月度支出规则，并算出年化合计。

上手三步
  1. 先在「周期账目」面板加几条规则（订阅 / 工资 / 房租等）。
  2. 切到「财务状况」首页，向下找到「现金流预测」+「月度订阅」并排两张卡。
  3. 切换 30 / 60 / 90 天，看累计净额折线如何变化。
P4 · 3.2

全局搜索 — ⌘K

任意页面按 ⌘K（Mac）或 Ctrl+K（Win），居中弹出搜索面板。同时搜 5 类：交易 / 账户 / 项目 / 分类 / 对手方。点交易 → 跳转流水页并预填搜索词；点账户/项目/分类/对手方 → 跳转设置页并切换到对应面板。

上手三步
  1. 任意页面按 ⌘K（顶部栏也有「搜索…」按钮）。
  2. 输入关键词 → 实时分组结果。
  3. 鼠标点结果即跳转 / 关闭。Esc 退出。
P4 · 3.4

仪表盘可定制

「财务状况」首页的 9 个 widget（KPI / 预算 / 储蓄目标 / 现金流预测 / 订阅 / 收入曲线 / 项目柱 / 工作生活 / 账户进度）可以单独显隐，也可以上下排序。布局保存到本地浏览器，下次访问保留。

上手三步
  1. 进「财务状况」，看右上角「自定义仪表盘」按钮。
  2. 点开右抽屉，每个 widget 三个按钮：上移 / 下移 / 显隐。
  3. 关掉抽屉立即生效。点「恢复默认顺序」可一键回滚。
P4 · 1.7

多币种 + 手动汇率

本位币默认 CNY。账户用别的币种（USD / HKD / EUR / JPY 等）时，账户卡片会显示原币 + ≈ ¥X CNY 折算，所有聚合（净资产、KPI）按汇率折算到本位币。

原币 $ 1,200.00 USD
≈ CNY ¥ 8,640.00
汇率 7.20
上手三步
  1. 进「财务设置 → 货币与单位」，第三张卡是「汇率」编辑器。
  2. 调整 USD / HKD / EUR / JPY 等汇率，或加新币种（如 GBP）。
  3. 建一个该币种的账户（账户卡片的 currency 字段），底部就会出现折算小卡。
本工具不依赖外部汇率 API（离线可用）。汇率需要手动维护，建议每月或交易前更新一次。
