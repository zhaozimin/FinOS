成员清单

types.ts: 财务领域共享契约，包含账户、类别、资金来源及软删除审计字段。
api/: HTTP 客户端边界，负责把页面操作映射到 Finance Node API。
pages/: 路由页面；账本页展示已删除流水，设置页保留已删除账户及资金影响。
components/: 可复用业务与基础 UI 组件。
lib/: 格式化、分析与页面辅助逻辑；其中资金流分析会把真实账本转换为图表可接受的无环模型。
store/: 页面范围和视图状态。

法则: 删除事实由后端保存；界面只能弱化呈现，不能把记录从用户视野中抹去。

[PROTOCOL]: 变更时更新此头部，然后检查 CLAUDE.md
